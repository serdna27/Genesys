Welcome to Genesys Project.

Genesys is a (System|Code Generator) written in scala. This tool is made to facilitate the creation of custom code that we as developer end up doing at any project. Some Genesys uses handlerbar](http://handlebarsjs.com/) templates to generate the code. Genesys supports 3 kinds of sources:

  • Database(all major rdbms)
  • Xml(Limited)
  • Json(Pending)
